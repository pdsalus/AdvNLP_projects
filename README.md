# AdvNLP Projects

This repository contains the graded lab work for the **Advanced Natural Language Processing (AdvNLP)** course.

## Repository Structure

All labs are organized into **separate subfolders**, each containing the notebooks, code, and any required data for the corresponding assignment.

Example structure:

AdvNLP_projects/
├── lab_01/
├── lab_02/
├── lab_03/
└── README.md
